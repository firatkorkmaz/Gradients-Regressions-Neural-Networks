# Gradients, Regressions, Neural Networks
A series of PyTorch implementations on MNIST, CIFAR-10 and CIFAR-100 datasets with Jupyter Notebook.

## General Information
This is a PyTorch project that contains 9 programs which start from simple Tensor examples to Convolutional Neural Networks trained with MNIST, CIFAR-10 and CIFAR-100 datasets.
1. **1_Numpy_Tensor.ipynb**: Numpy-tensor exercises.
2. **2_Gradient_Descent.ipynb**: Gradient descent example.
3. **3_Linear_Regression_1_Layer.ipynb**: Linear regression example.
4. **4_Logistic_Regression_1_Layer_MNIST.ipynb**: Logistic regression on MNIST dataset.
5. **5_Deep_Neural_Networks_2_Layers_MNIST.ipynb**: Deep neural networks on MNIST dataset.
6. **6_Convolutional_Neural_Networks_LeNet5_MNIST.ipynb**: A modified LeNet5 model on MNIST dataset.
7. **7_Convolutional_Neural_Networks_CIFAR10.ipynb**: Convolutional neural networks on CIFAR-10 dataset.
8. **8_Advanced_Neural_Networks_ResNet9_CIFAR10.ipynb**: Advanced neural networks on CIFAR-10 dataset.
9. **9_Advanced_Neural_Networks_ResNet9_CIFAR100.ipynb**: Advanced neural networks on CIFAR-100 dataset.
* **/ExternalNumbers/**: This folder has external numbers from 0 to 9, to further test the trained neural networks.
* **download.txt**: This file gives information about which dataset files to be downloaded before running the programs.

## Setup & Run
* Install the necessary libraries for the programs:
```
pip install -r requirements.txt
```
* Run Jupyter Notebook and open the Notebook files with it:
```
jupyter notebook
```
